# 功能方案

本目录用于沉淀 Attendance app 的具体业务功能方案。

当前一期还没有实现考勤业务，暂不拆具体功能文档。

后续建议按能力拆分：

| 文件 | 说明 |
|---|---|
| `attendance-rules.md` | 考勤规则、班次、迟到早退、缺卡规则 |
| `attendance-groups.md` | 考勤组和适用员工范围 |
| `attendance-records.md` | 打卡记录、异常状态、修正流程 |
| `attendance-approvals.md` | 补卡、外勤、请假、加班等审批入口 |
| `attendance-statistics.md` | 月度统计、部门统计、导出 |
